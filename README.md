# My Neovim Config Files
#### How to use  
Put into `~/.config/nvim/`

#### Useful Commands  
1. Lazy - opens up the Lazy package manager panel
2. TSUpdate/TSInstall - update/install treesitter parsers
3. Mason - opens up the LSP manager panel

#### Useful Keybinds  
1. `g d` - go to definition
2. `f k` - show error message
3. `j k (in insert mode)` - go back to normal mode (same as ESC)
4. `f12` - show references
5. `shift k` - hover tip
6. `space shift f` - format
7. `space t` - open terminal in a split pane below
8. `space f g` - opens up the grepping panel
9. `space n f` - opens up the Neotree File System in a floating window
10. `ctrl p` - opens up the file searching panel
11. `space h/j/k/l` - move cursor around panes in left/down/up/right directions
12. `alt h/v` - split pane in horiontal/vertical direction
13. `ctrl up/down/left/right` - resize current pane
14. `alt j/k (in visual mode)` - grab line and move down/up
